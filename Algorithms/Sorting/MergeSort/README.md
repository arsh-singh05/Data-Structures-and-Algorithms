# [`Merge Sort`](MergeSort.cpp)

## Algorithm
```
mergeSort(array, int start, int end):
    if (start >= end){
        return
    }
    mid = (start + end)/2
    mergeSort (arr, start, mid)
    mergeSort (arr, mid+1, end)
    merge (arr, start, mid, end)
```
